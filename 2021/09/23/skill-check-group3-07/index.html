[技能檢定]題組三 步驟7 前台院線片清單製作

發表於 2021-09-23 更新於 2022-01-20 屬於主題 網頁設計乙級 ， 題組三

題組三的三個前台功能中，院線片清單算是簡單的，主要就是照題意來列出四部院線片再加上分頁功能就可以了；

但是由於素材提供的css樣式和題目的不一樣，所以在這很容易就造成版面過擠不好看，但美觀的事可以晚點再做，我們先把功能完成，再回頭修一下 CSS 即可。

  1. 先在 index.php 的最上頭引入 base.php 檔
  2. 先前在切版時，我們把首頁中間的區塊切出去成為main.php，而這塊main.php其實包含了兩個功能，為了避免程式碼太多不好管理，所以我們再切成兩個獨立的檔案trailer.php及movie.php，這樣之後只要專注一個檔案一個功能就可以了。

front/main.php

front/trailer.php

front/movie.php

  1. 可以在前台顯示的院線片有以下條件：
    • 在後台管理中被設為 顯示 的影片
    • 上映日起算(含)的三天日期為觀影日
  2. 由於院線片清單需要製作分頁功能，一頁四部院線片，所以在製作時，我們會一同連分頁的程式碼一起撰寫。
  3. 這邊要小心的是因為條件為日期區間，所以需要撰寫一個稍微複雜的查詢條件。
  4. 刪除原本的 table 標籤區段，由於版型css已經不一樣了，使用表格來做排版反而麻煩，因此這邊我們建立一個新的容器，並使用flex及自己定義的css來做版面的簡單設定
  5. 利用迴圈將題意所需的圖片，資訊及連結分頁輸出至頁面上
  6. 撰寫基本的css來控制畫面的呈現，不用寫得太複雜太漂亮，能看就好，主要調整的是圖片的大小，位置排列的關係，字形大小，我會使用flex來做控制，並直接使用inline style的方式。
