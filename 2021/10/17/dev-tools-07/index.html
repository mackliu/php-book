[開發環境]常用快速鍵及打字練習

發表於 2021-10-17 更新於 2022-03-24 屬於主題 基礎課程 ， 開發環境與工具

做為一位長時間以程式碼輸入做為工具的程式人員，有些快鍵必須記住並常用，才能加速程式碼撰寫的速度，並且提升對於程式碼的掌握力。
此處的快速鍵以windows環境為主，Mac環境請參考相關文件及資料。

系統通用快速鍵

按鍵 功能
Ctrl + X 剪下選取的項目。
Ctrl + C (或 Ctrl + Insert) 複製選取的項目。
Ctrl + V (或 Shift + Insert) 貼上選取的項目。
Ctrl + Z 復原動作。
Alt + Tab 在開啟的應用程式之間切換。
Alt + F4 關閉使用中的項目，或結束使用中的應用程式。
F2 重新命名選取的項目。
Ctrl + A 選取文件或視窗中所有的項目。
Ctrl + R (或 F5) 重新整理使用中視窗。
Esc 停止或離開目前的工作。
Windows 標誌鍵+ Shift + S 抓取部分螢幕的螢幕擷取畫面。

Visual Studio Code 快速鍵

按鍵 功能
Ctrl + Shift + P 或 F1 開啟命令選擇區
Ctrl + Shift + S 另存新檔
Ctrl + K Ctrl + C 切換行註解
Ctrl + / 單行註解
Ctrl + K Ctrl + U 取消行註解
Ctrl + D 選取一段字詞
Ctrl + F 開啟搜尋
Ctrl + S 儲存
Alt + ↑ / ↓ 該行或多行向上或向下移動
Alt + Shift + ↑ / ↓ 向上或向下複製該行或多行
Alt + Ctrl + ↑ / ↓ 遊標向上或下垂直增加
Ctrl + Shift + ← / → 一次選取一個字詞(英文字)
